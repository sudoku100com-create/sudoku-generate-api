API Endpoints

GET

Dynamic Sudoku

https://www.sudoku100.com/sudoku-img

Generates a new random Sudoku puzzle on each request.

No parameters required
GET

By Difficulty

https://www.sudoku100.com/sudoku-img/{difficulty}

Generate Sudoku with specified difficulty level.

difficulty: beginner | easy | medium | hard | expert | extreme
GET

Custom Size & Format

https://www.sudoku100.com/sudoku-img?width={width}&format={format}

Get Sudoku with custom size and format.

width: 100-1000 format: png | webp | svg | jpg
GET

Image Embed

<iframe src="https://www.sudoku100.com/sudoku-img" width="400" height="400" style="border: none; overflow: hidden;"></iframe>

Embed Sudoku images directly into your web pages using iframe.

Supports all sudoku-img parameters
GET

Interactive Embed

https://www.sudoku100.com/embed/interactive

Embed an interactive Sudoku game on your website.

No parameters required
GET

Static Sudoku by ID

https://www.sudoku100.com/id/{id}

Get a specific Sudoku puzzle image by ID, ensuring the same puzzle every time.

id: 1-10000 width: 100-1000 format: png | webp | svg | jpg
